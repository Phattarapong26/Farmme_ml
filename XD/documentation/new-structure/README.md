## การเปลี่ยนแปลงหลัก

### 1. โครงสร้าง
- **เดิม**: 11 บท (แยกตามโมเดล)
- **ใหม่**: 5 บท (แยกตามกระบวนการวิจัย)

### 2. บทที่ 2
- ลบบทนำ
- แยกเป็นทฤษฎีและงานวิจัย
- เขียนเป็นย่อหน้าแทน bullets
- เพิ่มนิยามศัพท์เฉพาะ (ขยายความ)
- ไม่รวมการอธิบายการนำไปใช้

### 3. บทที่ 3
- สร้างใหม่ทั้งหมด
- อธิบายการดำเนินงานตามลำดับ
- ระบุว่าใช้ทฤษฎีใดจากบทที่ 2

### 4. บทที่ 4
- เน้นที่ Model A: Crop Recommendation
- มีตารางเปรียบเทียบโมเดล
- สรุปตามวัตถุประสงค์และขอบเขต

### 5. บทที่ 5
- มีการอภิปรายผลอย่างลึกซึ้ง
- อ้างอิงงานวิจัยที่สนับสนุน
- ระบุข้อจำกัดชัดเจน

### 6. ภาษา
- แปลงภาษาพูดเป็นภาษาเขียนทางวิชาการ
- เช่น "ควรปลูกเมื่อใด" → "การกำหนดช่วงเวลาการเพาะปลูก"

### 7. ประโยชน์
- ปรับให้วัดได้จริง
- ลบ "เพิ่มรายได้ 15-30%" (วัดไม่ได้)
- เพิ่ม "R² = 0.47, MAE = 13.31" (วัดได้)

### 8. รูปแบบ
- แปลง bullets เป็นย่อหน้าในส่วนที่เหมาะสม
- รักษา bullets เฉพาะรายการขั้นตอนและตาราง
